CSS selectors : 1. element selector : The element selector selects all elements with the specified element name. 2. element,element :To style several elements with the same style, separate each element name with a comma. example : h1, p { background-color: yellow; } here we want to apply the same properties to both h1 and p element. 3 element element: The element element selector is used to select elements inside elements. example : div p { background-color: yellow; } that means all the p inside div
